# mtg.py
## Описание
mtg.py - Minecraft Table Generator. То есть он генерирует таблички из Minecraft c текстом.
## Пример
### Код:
```python
import mtg

mtg.generate_table("Python - Топ!", mtg.ColorRGBA("green")).show()
```
